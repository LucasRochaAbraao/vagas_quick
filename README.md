Repositório para RESTAPI das vagas no site quick.com.br

## utilização
Repositório para RESTAPI das vagas no site quick.com.br

- gunicorn --workers=2 --name=vagas_quick 'app:app'

## instalação
- sudo dnf install python3-gunicorn python3-flask | sudo apt install python3-gunicorn python3-flask
